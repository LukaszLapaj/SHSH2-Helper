# SHSH2-Helper
Simple Python Script to automate SHSH2 Blobs download and store on Windows.

# Install
```
git clone https://github.com/LukaszLapaj/SHSH2-Helper.git
```

# Configuration
Go to `helper.py` and set necessary information:
```
ECID = ''
MODEL = ''
BOARDCONFIG = ''
bloobsPerVersion = 3
```

# Use
```
python helper.py
```
